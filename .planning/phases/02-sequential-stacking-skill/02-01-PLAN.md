Create `ps_bed/skills/stack_n.py` with a `StackNSkill` class that inherits from `PickPlaceSkill`.

**Class structure:**

1. **Module-level:** Import `logging`, create logger: `logger = logging.getLogger("ps_bed.skills.stack_n")`

2. **`StackNSkill(PickPlaceSkill)`** inherits all primitives (`_setup_planner`, `_move_to_pose_with_screw`, `_actuate_gripper`, `_follow_path`) without modification.

3. **`_pick_and_stack(self, env, planner, cube_to_pick, target_cube, step_index, total_steps)`** method:
   - Computes grasp pose from OBB of `cube_to_pick` using `get_actor_obb()` + `compute_grasp_info_by_obb()` + `agent.build_grasp_pose()` (same pattern as existing `PickPlaceSkill.solve()`)
   - Searches for collision-free grasp orientation using the same angle search pattern (angles from 0 to 2pi/3, alternating positive/negative, in pi/2 increments). If all angles fail, return None with `failure_reason="grasp_plan_failed"`
   - Reach phase: approach from 0.05m behind grasp pose. Abort on plan failure.
   - Grasp phase: move to grasp pose, close gripper. **Verify grasp with `raw.agent.is_grasping(cube_to_pick)`** per user decision. If not grasping, abort with `failure_reason="grasp_verification_failed"`.
   - Lift phase: Use **dynamic lift height** per user decision. Formula: `lift_z = max(0.1, (step_index + 2) * cube_height + 0.05)` where `cube_height = (raw.cube_half_size[2] * 2).item()`. This scales with stack progress — for step 0 (stacking cube_1 on cube_0), lift_z = max(0.1, 2*0.04+0.05) = 0.13; for step 2 (cube_3 on stack of 3), lift_z = max(0.1, 4*0.04+0.05) = 0.21. Abort on plan failure.
   - Place phase: Compute dynamic target from `target_cube.pose * sapien.Pose([0, 0, cube_height])`. Compute offset from cube's current position to goal, apply to lifted pose. Abort on plan failure.
   - Release phase: Open gripper.
   - **Settling steps** per user decision: After releasing, execute additional no-op actions (10 steps) to let physics settle before next pick. Use `_actuate_gripper(env, planner, self.GRIPPER_OPEN, steps=10)`.
   - **Placement verification** per user decision: After settling, check `cube_to_pick.pose.p[..., 2]` is within threshold of expected z-position (`target_cube.pose.p[..., 2] + cube_height`). Threshold: 0.01m (half a cube height). If out of range, return None with `failure_reason="placement_check_failed"`.
   - On success, return the last `(obs, reward, terminated, truncated, info)` tuple from the release/settling step.
   - On any failure, return `None`.
   - Log progress with `logger.info(f"Picking cube_{i+1}...")`, `logger.info(f"Placing on cube_{i}...")`, `logger.info(f"Step {step_index+1}/{total_steps} complete")`.
   - On failure, log with `logger.warning(f"Step {step_index+1}/{total_steps} FAILED: {failure_reason} for cube_{cube_index}")`.

4. **`solve(self, env, seed=None)`** method:
   - Reset env with seed.
   - Assert control mode is `pd_joint_pos` or `pd_joint_pos_vel`.
   - Call `self._setup_planner(env)`.
   - Get `raw = env.unwrapped`, `cubes = raw.cubes`, `n = len(cubes)`.
   - Loop `i` from 0 to `n-2`: pick `cubes[i+1]`, place on `cubes[i]`.
   - Call `_pick_and_stack(env, planner, cubes[i+1], cubes[i], step_index=i, total_steps=n-1)`.
   - If result is None (failure):
     - Get current observation: `obs, _, _, _, base_info = env.step(np.hstack([current_qpos, self.GRIPPER_OPEN]))` (one no-op step to get valid obs). Actually, simpler: use the last valid step result. But per user decision, always return valid obs. So: read current qpos, step once with no-op to get fresh obs/info.
     - Build info dict: start with `raw.evaluate()`, add `"cubes_stacked": i` (number successfully stacked so far) and `"failure_reason": reason_string`.
     - Return `(obs, 0.0, False, False, info)`.
   - If loop completes successfully:
     - Return the last `(obs, reward, terminated, truncated, info)` from the final `_pick_and_stack` call, but augment info with `"cubes_stacked": n - 1`.
   - Log summary: `logger.info(f"Stacking complete: {n-1}/{n-1} cubes stacked")` on success, or `logger.warning(f"Stacking aborted: {i}/{n-1} cubes stacked")` on failure.

**Critical constraints:**
- Do NOT modify `ps_bed/skills/pick_place.py` at all. `PickPlaceSkill` must remain unchanged for backward compatibility.
- Use `_sapien_to_mplib_pose` from pick_place module (it's a module-level function, import it).
- Handle SAPIEN batched tensors: `.cpu().numpy()[0]` for position, `.item()` for scalars.
- `is_grasping` returns a batched tensor — use `bool(is_holding.item())` if tensor.
- The `_move_to_pose_with_screw` return convention: returns -1 on failure, tuple on success, dict on dry_run.
