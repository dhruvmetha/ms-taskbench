Create `ps_bed/envs/stack_n_cube.py` with `StackNCubeEnv` class extending `BaseEnv` directly (NOT StackCubeEnv -- it hardcodes 2-cube assumptions in every method, per research decision).

Register with `@register_env("StackNCube-v1", max_episode_steps=250)`.

**Constructor (`__init__`):**
- Accept `num_cubes: int = 3` kwarg, `robot_uids="panda_wristcam"`, `robot_init_qpos_noise=0.02`
- Store `self.num_cubes` and `self.robot_init_qpos_noise` BEFORE calling `super().__init__()`
- Set `SUPPORTED_ROBOTS = ["panda_wristcam", "panda", "fetch"]`
- Set `SUPPORTED_REWARD_MODES = ["sparse", "none"]` (no dense reward -- motion planner doesn't use it, per research)
- Type hint `agent: Union[Panda, Fetch]`

**`_default_sim_config` property:**
- Return `SimConfig(scene_config=SceneConfig(solver_position_iterations=20, solver_velocity_iterations=4))` for physics stability with tall stacks (per research decision to use 20+ solver iterations)

**`_load_scene(self, options: dict)`:**
- Create `self.cube_half_size = common.to_tensor([0.02] * 3, device=self.device)`
- Build table scene with `TableSceneBuilder(env=self, robot_init_qpos_noise=self.robot_init_qpos_noise)`
- Build N cubes in a loop using `actors.build_cube()` with distinct colors from CUBE_COLORS list (red, green, blue, yellow, magenta, cyan), storing in `self.cubes` list
- Use `initial_pose=sapien.Pose(p=[i * 0.1, 0, 0.1])` for initial dummy poses
- Name cubes `f"cube_{i}"`

**`_initialize_episode(self, env_idx: torch.Tensor, options: dict)`:**
- Use `b = len(env_idx)` for batch size (NOT hardcoded to 1)
- Initialize table scene: `self.table_scene.initialize(env_idx)`
- Create xyz tensor `(b, 3)` with z=0.02 (half cube height above table)
- Use `randomization.UniformPlacementSampler` with bounds `[[-0.1, -0.2], [0.1, 0.2]]`
- Compute radius as `torch.linalg.norm(torch.tensor([0.02, 0.02])) + 0.001`
- Loop through cubes: sample XY via `sampler.sample(radius, 100, verbose=False)`, generate random Z-rotation quaternion via `randomization.random_quaternions(b, lock_x=True, lock_y=True, lock_z=False)`, set pose via `Pose.create_from_pq(p=xyz.clone(), q=qs)` -- use `.clone()` for all but the last cube to avoid shared memory
- Do NOT apply the xy group offset that StackCubeEnv uses -- per research, dropping it prevents cubes spawning near table edges for N=6, and the sampler provides sufficient randomness on its own

**`evaluate(self)`:**
- Get positions for all cubes: `pos = [cube.pose.p for cube in self.cubes]`
- Initialize `all_pairs_stacked`, `all_static` as True tensors, `any_grasped` as False tensor, all shape `(self.num_envs,)` on `self.device`
- Compute thresholds: `xy_thresh = torch.linalg.norm(self.cube_half_size[:2]) + 0.005`, `z_target = self.cube_half_size[2] * 2` (one full cube height)
- Loop from i=1 to N-1: check offset between pos[i] and pos[i-1]:
  - `xy_ok`: XY distance <= xy_thresh
  - `z_ok`: abs(Z offset - z_target) <= 0.005
  - AND into `all_pairs_stacked`
  - Check `self.cubes[i].is_static(lin_thresh=1e-2, ang_thresh=0.5)` AND into `all_static`
  - Check `self.agent.is_grasping(self.cubes[i])` OR into `any_grasped`
- Also check cube[0]: is_static AND into all_static, is_grasping OR into any_grasped
- `success = all_pairs_stacked & all_static & (~any_grasped)`
- Return dict with keys: `all_pairs_stacked`, `all_static`, `any_grasped`, `success` (as `.bool()`)

**`_get_obs_extra(self, info: dict)`:**
- Always include `tcp_pose=self.agent.tcp.pose.raw_pose`
- If `"state" in self.obs_mode`: add per-cube pose (`cube_{i}_pose`), tcp-to-cube position (`tcp_to_cube_{i}_pos`), and pairwise adjacent cube distances (`cube_{i-1}_to_cube_{i}_pos`)

**`compute_sparse_reward(self, obs, action, info)`:**
- Return `info["success"].float()`

**Verification script** (run inline, do not create a permanent file):

Write and execute a verification script that tests all four ENV requirements:

```python
import gymnasium as gym
import torch
import ps_bed.envs  # trigger registration

errors = []

# ENV-01: gym.make with num_cubes=N for N in [2, 3, 4, 5, 6]
for n in [2, 3, 4, 5, 6]:
    env = gym.make(
        "StackNCube-v1",
        num_cubes=n,
        obs_mode="state",
        num_envs=1,
        sim_backend="cpu",
        render_mode="rgb_array",
    )
    obs, info = env.reset(seed=42)

    # Check correct number of cubes
    unwrapped = env.unwrapped
    if len(unwrapped.cubes) != n:
        errors.append(f"N={n}: expected {n} cubes, got {len(unwrapped.cubes)}")

    # ENV-02: Check cubes don't overlap (pairwise XY distance > 2*diagonal)
    positions = [cube.pose.p for cube in unwrapped.cubes]
    min_dist = torch.linalg.norm(torch.tensor([0.02, 0.02])) * 2
    for i in range(n):
        for j in range(i + 1, n):
            xy_dist = torch.linalg.norm(
                positions[i][..., :2] - positions[j][..., :2]
            )
            if xy_dist < min_dist * 0.9:  # small margin
                errors.append(
                    f"N={n}: cubes {i},{j} overlap (dist={xy_dist:.4f} < {min_dist:.4f})"
                )

    # ENV-03: evaluate() returns correct structure
    eval_result = unwrapped.evaluate()
    for key in ["all_pairs_stacked", "all_static", "any_grasped", "success"]:
        if key not in eval_result:
            errors.append(f"N={n}: evaluate() missing key '{key}'")

    # ENV-04: max_episode_steps = 250
    spec = env.spec
    if spec and spec.max_episode_steps != 250:
        errors.append(
            f"N={n}: max_episode_steps={spec.max_episode_steps}, expected 250"
        )

    env.close()
    print(f"N={n}: OK ({len(unwrapped.cubes)} cubes)")

if errors:
    print("\nFAILURES:")
    for e in errors:
        print(f"  - {e}")
    raise AssertionError(f"{len(errors)} verification failures")
else:
    print("\nAll ENV-01 through ENV-04 requirements verified.")
```
